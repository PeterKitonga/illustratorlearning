## About Lesson 17

### Brief
In this lesson, I learned about the various symbol tools in illustrator. These tools are used to modify symbols. The symbol tools include:
- Symbol Sprayer Tool
- Symbol Shifter Tool
- Symbol Scruncher Tool
- Symbol Sizer Tool
- Symbol Spinner Tool
- Symbol Stainer Tool
- Symbol Screener Tool
- Symbol Styler Tool
